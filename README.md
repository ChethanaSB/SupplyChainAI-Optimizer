# ChainMind — Logistics Intelligence Platform

**ChainMind** is a production-ready, AI-enhanced logistics platform designed for high-precision supply chain optimization, demand forecasting, and disruption resilience. Built with a premium industrial dark aesthetic, it integrates real-time global data streams into a cognitive optimization engine.

## 🚀 Key Features

- **Demand Forecasting**: Multi-horizon predictions using Temporal Fusion Transformers (TFT) with P10/P50/P90 confidence intervals.
- **Dynamic Routing**: Multi-objective Vehicle Routing Problem (VRP) solver using Google OR-Tools, balancing Cost, Lead Time, and CO₂ Emissions.
- **Disruption Radar**: Real-time signal intelligence from NewsAPI, Port Congestion data, and Weather patterns, processed through HuggingFace NER and sentiment models.
- **AI Playbooks**: Instant, context-aware recovery plans generated by Google Gemini for 5 distinct disruption scenarios.
- **Resilience Testing**: Monte Carlo simulation engine and a "Crisis Simulator" to stress-test your network against port closures and carrier crunches.
- **Live KPI Dashboard**: Real-time tracking of 8 core KPIs compared against a statistical reorder-point baseline.

## 🛠 Tech Stack

- **Backend**: FastAPI, SQLAlchemy (PostgreSQL/Neon), Redis Streams (Upstash), Pydantic.
- **AI/ML**: PyTorch Forecasting, Google Gemini, Scikit-Learn (Isolation Forest), Stable-Baselines3 (PPO RL).
- **Optimization**: Google OR-Tools (CP-SAT), PuLP (MILP).
- **Frontend**: Next.js 14, Tailwind CSS, Recharts, Leaflet, Axios, Lucide Icons.
- **Monitoring**: MLflow for experiment and model tracking.

## 📋 Prerequisites & Setup

You will need the following API keys to enable full functionality:

1. **Google Gemini**: [API Key](https://aistudio.google.com/app/apikey) (`GEMINI_API_KEY`)
2. **Neon PostgreSQL**: [Neon.tech](https://console.neon.tech/) (`DATABASE_URL`)
3. **Upstash Redis**: [Upstash.com](https://console.upstash.com/) (`REDIS_URL`)
4. **Alpha Vantage**: [Finance Key](https://www.alphavantage.co/support/#api-key) (`ALPHA_VANTAGE_KEY`)
5. **NewsAPI**: [News Key](https://newsapi.org/register) (`NEWSAPI_KEY`)

### Quick Start (Local Development)

1. **Clone and Install Backend**:
   ```bash
   cd chainmind/backend
   pip install -r requirements.txt
   uvicorn main:app --reload
   ```

2. **Clone and Install Frontend**:
   ```bash
   cd chainmind/frontend
   npm install
   npm run dev
   ```

3. **Docker Deployment**:
   ```bash
   docker-compose up --build
   ```
